# 🧮 Kalkulyator

Python dasturlash tilida yozilgan oddiy kalkulyator.

## Imkoniyatlari

* ➕ Qo'shish
* ➖ Ayirish
* ✖️ Ko'paytirish
* ➗ Bo'lish
* 0 ga bo'lishdan himoyalangan
* Noto'g'ri kiritilgan ma'lumotlarni tekshiradi

## Talablar

* Python 3.10 yoki undan yangi versiya

## Ishga tushirish

```bash
python kalkulyator.py
```
