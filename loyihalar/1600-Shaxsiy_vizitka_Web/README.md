# 🚀 Premium Portfolio Vizitka

Bu loyiha zamonaviy **Glassmorphism** (shishasimon) dizayn uslubida tayyorlangan shaxsiy vizitka-sayt hisoblanadi. Loyiha o'quv topshirig'i doirasida yaratilgan bo'lib, HTML va CSS imkoniyatlarini maksimal darajada ko'rsatib beradi.

## ✨ Xususiyatlari
- 💎 **Glassmorphism:** Xira va shaffof elementlar uyg'unligi.
- 🌈 **Dynamic Background:** Orqa fonda harakatlanuvchi rangli "Blob" animatsiyalari.
- 📱 **Responsive:** Barcha turdagi qurilmalar (mobil, planshet, kompyuter) uchun moslashtirilgan.
- ⚡ **Animations:** Sahifa yuklanganda elementlarning silliq paydo bo'lishi.
- 🎨 **Modern UI:** Toza va chiroyli foydalanuvchi interfeysi.

## 🛠 Texnologiyalar
- **HTML5:** Sayt strukturasi va mazmuni uchun.
- **CSS3:** Dizayn, animatsiyalar va glassmorphism effektlari uchun.
- **Google Fonts:** Zamonaviy shriftlar (Poppins).

## 📂 Fayllar strukturasi
- `index.html` - Saytning asosiy kodi va matnlari.
- `style.css` - Dizayn sozlamalari va animatsiya qoidalari.
- `README.md` - Loyiha haqida umumiy ma'lumot (ushbu fayl).

## 🚀 Qanday ishga tushiriladi?
1. Ushbu loyihani brauzerda ochish uchun shunchaki tahrirlagichdagi **"Run"** tugmasini bosing.
2. Agar o'z kompyuteringizda ishlatmoqchi bo'lsangiz, `index.html` va `style.css` fayllarini bitta papkaga joylang va HTML faylni brauzer orqali oching.
