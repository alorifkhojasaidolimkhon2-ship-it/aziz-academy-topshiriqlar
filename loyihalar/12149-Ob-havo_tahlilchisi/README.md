# 🌤️ Ob-havo tahlilchisi

Python dasturlash tilida yozilgan konsol dasturi. Dastur haroratlarni saqlaydi va ularni tahlil qilish imkonini beradi.

## Imkoniyatlari

- 🌡️ Harorat qo'shish (`add`)
- 📊 Statistikani ko'rsatish (`stat`)
- 📈 Trendni aniqlash (`trend`)
- ⚠️ Ogohlantirish va tekshirish
- 📄 Yakuniy hisobot (`report`)
- 🚪 Dasturdan chiqish (`exit`)

## Talablar

- Python 3.10 yoki undan yuqori

## Ishga tushirish

```bash
python main.py
```

## Buyruqlar

| Buyruq | Tavsif |
|--------|--------|
| `add <son>` | Haroratni ro'yxatga qo'shadi |
| `stat` | Statistikani chiqaradi |
| `trend` | Harorat trendini ko'rsatadi |
| `report` | Yakuniy hisobotni chiqaradi |
| `exit` | Dasturdan chiqadi |

## Misol

```text
add 25
Qabul: 25

add -5
Qabul: -5

stat

trend

report

exit
```
